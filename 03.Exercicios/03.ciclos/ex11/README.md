# Ex 11 - Ciclos
Escreva um programa em C que solicite n (solicitar quantos) números ao utilizador (inteiros). No final deve apresentar os seguintes resultados:
- Média dos números (real)
- soma dos números
- Maior inserido
- Menor inserido

Exemplo:
```text
Insira n1->23
Insira n2->-4
...
Insira n10->2

Resultados
Média: 15.7
Soma: 157
Maior: 23
Menor: -4

```
