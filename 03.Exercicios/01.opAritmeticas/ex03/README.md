# Op Aritméticas - Ex03
Efetue um programa em C que  escreva no ecrã o valor da área de um quadrado de lado l. O valor de l deve ser solicitado ao utilizador

- Instruções
    - Criar um ficheiro ex03.c
    - Compilar com o comando `gcc ex03.c -o ex03.out`
    - Executar com `./ex03.out`

- Output esperado
```shell
Insira l->4
O quadrado tem uma area de: 16

```
